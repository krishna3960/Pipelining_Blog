# Pipelining and Data Hazards Blog

An interactive educational blog that explains **pipelining** and **data hazards** in computer architecture. This project aims to help students and learners understand how instruction pipelining works and data hazards that can occur (RAW). This was created as an interactive experience with visual explanations.


### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- npm (comes with Node.js)

### Installation

```bash
npm install
```

### Available Scripts

#### `npm start`

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

#### `npm run build`

Builds the app for production to the `build` folder.

#### `npm test`

Launches the test runner in interactive watch mode.
